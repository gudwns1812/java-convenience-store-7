## 기능 목록

- 입출력 요구사항
    - [x]  사용자에게 현재 시스템의 물품 현황을 출력한다.
    - [ ]  사용자가 구매할 상품명과 수량을 입력받는다.
    - [ ]  프로모션 적용 상품의 수량을 덜 가져왔을 경우 안내 메세지를 출력한다.
    - [ ]  프로모션 재고가 부족해 일부 수량을 정가로 결제해야 하는 경우 안내 메세지를 출력한다.
    - [ ]  멤버십 할인 적용 여부를 입력 받는다.
    - [ ]  사용자에게 구매 내역, 증정 상품, 금액 정보(총구매액, 행사할인, 멤버십 할인,내실돈)를 출력한다.
    - [ ]  사용자에게 추가 구매 여부를 입력받고 추가 구매할 경우 다시 물품 현황을 출력한다.
- 기능 요구사항
    - [ ]  프로모션 상품이 있을 경우 프로모션 제품 부터 우선 차감한다.
    - [ ]  프로모션 상품을 결제할 경우 프로모션 조건과 비교해 물품 수량을 체크한다.
    - [ ]  프로모션 재고가 부족할 경우 사용자에게 일반 재고로 할것인지 물어본다음 일반 재고에서 차감한다.
    - [ ]  멤버십 할인은 프로모션 미적용 상품에 대해서 30% 적용한다.
    - [ ]  프로모션 재고일지라도 재고가 부족하거나 사용자가 재고를 적게 살 경우 프로모션 미적용 상품으로 취급한다.
    - [ ]  멤버십 할인의 한도는 8,000원이다.
- 예외 사항
    - [ ]  사용자가 정해진 입출력 규칙이 아닌 다르게 입력할경우 예외를 발생시킨다.
    - [ ]  상품명이 없을경우 존재하지 않는 상품입니다. 다시 입력해 주세요. 라는 문구와 함께 예외를 발생시킨다.
    - [ ]  구매 수량이 재고를 초과할 경우 수량 초과입니다. 다시 입력해 주세요. 라는 문구와 함께 예외를 발생시킨다.
